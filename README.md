### <u>Run Locally</u>

Clone the project

```
https://github.com/rahulr41180/city-country
```

Install npm packages

npm install

Run the server

```
npm run server
or
npx json-server db.json --port 8080 --watch
```

Run the Frontend

```
npm start
```

Redirect to

```
http://localhost:3000
```
